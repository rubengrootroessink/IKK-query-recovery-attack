Massimo,

You will been to get the Lucene distribution that contains build.xml
and you will also have to make sure that you have Ant installed and set
up (http://jakarta.apache.org/ant).
Then you can run 'ant compile', which will take care of converting .jj
files to .java files and compiling them into .class files.
