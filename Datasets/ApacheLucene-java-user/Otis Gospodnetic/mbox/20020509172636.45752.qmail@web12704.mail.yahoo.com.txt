I don't think there is a human-readable file in the index that will get
you the number of indexed documents.  If you find it, let me know, I'd
like to know about it and add it to the FAQ.

files:

_6p.f3:    MP32, Mono
_6p.fdt:   data
_6p.fdx:   data
_6p.fnm:   data
_6p.frq:   data
_6p.prx:   data
