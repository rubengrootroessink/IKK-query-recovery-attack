SUBJECT: ANNOUNCEMENT: Release of Lucene 1.2
DATE: 13 Jun 2002

The Lucene Team is proud to announce the release of Lucene 1.2. This is the
first production release of Lucene since it moved to the Apache project.
This release contains many features and bug fixes over the previous 1.0.2
release - see CHANGES.txt for details. Jakarta Lucene is a high-performance,
full-featured text search engine written entirely in Java. Binary and source
distributions are available
http://jakarta.apache.org/builds/jakarta-lucene/release/v1.2/.
