Sorry - kind of my fault. When I fixed this to use maxDocCharsToAnalyze, I didn't set a default
other than 0 because I didn't really count on this being used beyond how it is in the Highlighter
- which always sets maxDocCharsToAnalyze with it's default.
 
You've got to explicitly set it higher than 0 for now.

Feel free to create a JIRA issue and we can give it's own default greater than 0.
